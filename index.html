Github Pull Request jQuery Widget

Github doesn't make it all that easy to get a list of the pull requests you have submitted to other projects. However, you can get the information from the API in a slightly roundabout way. This plugin does exactly that and makes listing your pull requests easy.

Demo

You can see it in action on the right.

Why would I need this?

On Github, a lot of work is happening in pull requests. After my pull request is merged (or rejected *sniff*) I usually delete my own fork as to not litter my own account. However, I would like the pull requests to still be visible somewhere because being able to show open source contributions usually makes job interviews a lot easier.

Usage

  1. Download the latest archive and unpack
  2. Add the plugin to your document's head 
    <script src="pull-request-widget.js"></script>
  3. Optionally, also add the CSS 
    <link rel="stylesheet" href="pull-request-widget.css"/>
  4. Add an element with your username 
    <div id="pull-requests" data-github-user="lenniboy"></div>
  5. Instantiate the widget 
    $(document).ready(function(){
  $("#pull-requests").pullRequests();
});

Caching

Since each browser has only 5000 API calls per day and this widget needs to make a lot of requests, the pull request data is cached in localStorage for 60 minutes.
